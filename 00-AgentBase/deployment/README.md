# Deployment — 部署

> 知识地图模块:**Deployment** — 部署模式、环境隔离、CI/CD、密钥管理

## 这个目录放什么

关于 Agent 部署的认知(运维视角):
- Agent 部署和普通应用部署的区别
- 三种典型部署模式(本地交互式 / 服务端长驻 / 嵌入式集成)
- 环境隔离(开发/预发/生产)
- CI/CD 中的 Agent
- 部署的安全考虑
- 部署的可观测性
- 版本管理

> **与 Runtime 模块的边界**:Runtime 模块(`runtime/`)讲的是某个具体 Runtime 的安装和配置,本目录讲的是 Agent 整体部署的模式选择、环境治理和生命周期。
