## 这是什么

Claude Code 是 Anthropic 开发的 Agentic 编码工具。它不是一个简单的代码补全工具,而是一个能**理解整个代码库、跨多文件编辑、执行命令、运行测试、管理 Git 工作流**的 Agent Runtime。

核心特征:

- **多表面运行**:终端 CLI、VS Code、JetBrains、桌面应用、Web 浏览器、移动端——同一引擎,不同入口
- **Agentic 循环**:不是一问一答,而是自主规划、执行、验证的多步任务循环
- **上下文感知**:读取整个代码库,理解架构、依赖、约定
- **工具丰富**:内置 Bash、文件读写、搜索、Web 等工具,通过 MCP 无限扩展
- **安全分层**:六种权限模式,从手动确认到自动审批
- **可定制**:CLAUDE.md 记忆、Skills 技能、Hooks 钩子、Subagents 子代理四大定制机制

## 类比

大多数 AI 编码工具像"计算器"——你输入公式,它给结果。Claude Code 像"搭档"——你描述目标,它理解上下文、规划方案、写代码、跑测试、提交 PR,全程可见可控。

## 核心特性

| 特性 | 说明 |
|------|------|
| **多表面架构** | 同一引擎覆盖终端、VS Code、JetBrains、桌面应用、Web、移动端。CLAUDE.md、设置、MCP 服务器跨表面通用 |
| **Agentic 循环** | 自主规划→执行→验证的多步任务循环。支持 `/plan` 模式先分析后编码 |
| **CLAUDE.md 记忆** | Markdown 文件持久化项目指令,每次会话自动加载。支持多级作用域和 `@import` 导入 |
| **Auto Memory** | Claude 自动从交互中学习,保存构建命令、调试经验、用户偏好,无需手动编写 |
| **Skills 技能系统** | 可复用的工作流打包为 `/skill-name` 斜杠命令。兼容 agentskills.io 开放标准 |
| **Subagents 子代理** | 生成隔离子代理处理并行任务,每个有独立上下文窗口和工具权限 |
| **Hooks 钩子** | 20+ 生命周期事件,在工具调用前后、会话开始结束等节点执行自定义逻辑 |
| **MCP 集成** | 通过 Model Context Protocol 连接数百种外部工具和数据源。支持 stdio/HTTP/SSE/WebSocket 四种传输 |
| **六种权限模式** | default(手动)→ acceptEdits(自动编辑)→ plan(只读分析)→ auto(智能自动)→ dontAsk(预批准)→ bypassPermissions(全跳过) |
| **CLI 可组合** | 遵循 Unix 哲学,支持管道、脚本、CI/CD 自动化 |
| **定时任务** | Routines(云端)和 Desktop 定时任务(本地),实现无人值守自动化 |
| **Agent SDK** | 基于 Claude Code 工具构建自定义 Agent,完全控制编排、工具访问和权限 |

## 多表面架构

```
┌─────────────────────────────────────────────────┐
│              Claude Code 引擎                     │
│   (CLAUDE.md + Settings + MCP + Tools)           │
└──────────────────────┬──────────────────────────┘
                       │
    ┌──────┬───────┬───┴───┬────────┬────────┐
    │      │       │       │        │        │
  终端    VS Code  JetBrains  桌面    Web    移动端
  CLI    扩展      插件      应用   浏览器   iOS
```

所有表面连接同一引擎,因此 CLAUDE.md 文件、设置和 MCP 服务器跨表面通用。

## 四大定制机制

| 机制 | 解决什么 | 加载时机 | 详见 |
|------|----------|----------|------|
| **CLAUDE.md** | 持久指令(编码标准、架构决策) | 每次会话启动 | [记忆系统](./06-记忆系统.md) |
| **Skills** | 按需流程(部署、审查、生成) | 调用时加载 | [技能系统](./05-技能系统.md) |
| **Hooks** | 硬约束执行(编辑后格式化) | 生命周期事件触发 | [Hooks 系统](./09-Hooks系统.md) |
| **Subagents** | 隔离分工(探索、审查、调试) | 任务委派时生成 | [子代理系统](./07-子代理系统.md) |

## 内置工具概览

| 工具 | 作用 | 需要权限 |
|------|------|----------|
| `Bash` / `PowerShell` | 执行 Shell 命令 | 是 |
| `Read` / `Edit` / `Write` | 读写编辑文件 | 是(Edit/Write) |
| `Grep` / `Glob` | 搜索文件内容和文件名 | 否 |
| `Agent` | 生成子代理 | 否 |
| `WebFetch` / `WebSearch` | 网页获取和搜索 | 是 |
| `LSP` | 语言服务器代码智能 | 否 |
| `Monitor` | 后台命令监控 | 是 |
| `Task*` | 任务管理(TaskCreate/List/Update) | 否 |
| `Skill` | 执行技能 | 是 |

> 工具通过权限规则和 Hooks 配置。详见 [工具系统](./04-工具系统.md)。

## 核心命令

| 命令 | 作用 |
|------|------|
| `claude` | 启动交互式会话 |
| `claude "query"` | 带初始提示启动 |
| `claude -p "query"` | 非交互式查询后退出 |
| `cat file \| claude -p "query"` | 管道处理 |
| `claude -c` | 继续最近的对话 |
| `claude -r "<session>" "query"` | 按 ID 恢复会话 |
| `claude update` | 更新到最新版本 |
| `claude mcp` | 配置 MCP 服务器 |
| `claude doctor` | 诊断问题 |
