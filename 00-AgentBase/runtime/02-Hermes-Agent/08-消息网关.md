## 这是什么

消息网关是一个**单一后台进程**，连接所有已配置的消息平台，处理会话、运行 cron 任务、投递语音消息。

可以把它想象成一个"翻译中心"：左边是各种聊天软件（Telegram、Discord、微信……），右边是你的 AI Agent。无论消息从哪个平台进来，网关都把它统一翻译成 Agent 能理解的格式，再把 Agent 的回复翻译回对应平台的格式投递出去。

## 支持平台（23+）

Telegram、Discord、Slack、WhatsApp、Signal、SMS、Email、Home Assistant、Mattermost、Matrix、DingTalk（钉钉）、Feishu/Lark（飞书）、WeCom（企微）、Weixin（微信）、BlueBubbles（iMessage）、QQ、Yuanbao、Microsoft Teams、LINE、ntfy、Raft、IRC、Google Chat

## 架构

每个平台适配器接收消息，通过 per-chat 会话存储路由，分发给 AIAgent 处理。网关还运行 cron 调度器，每 60 秒检查一次到期任务。

│                                                   │

│  │ Telegram │ │ Discord  │ │  Slack   │  ...     │
│  │ Adapter  │ │ Adapter  │ │ Adapter  │          │
│  └────┬─────┘ └────┬─────┘ └────┬─────┘          │

│       └────────────┼────────────┘                 │

## 网关命令

```bash
hermes gateway              # 前台运行
hermes gateway setup        # 交互式配置
hermes gateway install      # 安装为用户服务
hermes gateway start/stop/status  # 服务管理
```

## 会话管理

会话跨消息持久化，直到被重置。重置策略：

| 模式 | 说明 |
|---|---|
| none | 从不自动重置（默认） |
| daily | 每天特定时间重置 |
| idle | N 分钟不活动后重置 |
| both | 先触发的为准 |

支持 per-platform 覆盖，例如 Telegram 用 daily、Discord 用 idle。

## 用户授权

默认拒绝所有未在允许列表或未通过 DM 配对的用户。

**允许列表**：设置 `TELEGRAM_ALLOWED_USERS`、`DISCORD_ALLOWED_USERS` 等，或用 `GATEWAY_ALLOWED_USERS` 全局允许。

**DM 配对**：未知用户收到一次性配对码，bot 拥有者通过以下命令批准：

```bash
hermes pairing approve <platform> <code>
```

配对规则：

| 项目 | 值 |
|---|---|
| 配对码长度 | 8 字符 |
| 有效期 | 1 小时 |
| 限速 | 10 分钟内不可重复请求 |
| 最大待处理数 | 3 个 |

## Admin vs 普通用户

| 角色 | 权限 |
|---|---|
| Admin | 完全访问，可运行所有斜杠命令 |
| 普通用户 | 受限访问，只能运行显式启用的命令 |

通过 `/whoami` 查看当前权限。

## 中断 Agent

发送任何消息可中断正在工作的 Agent：

- 进行中的终端命令立即终止
- 工具调用被取消
- 多条消息合并为一个提示
- `/stop` 命令不带后续消息，仅中断

忙碌输入模式：

| 模式 | 说明 |
|---|---|
| interrupt | 收到新消息立即中断当前任务（默认） |
| queue | 新消息排队，等待当前任务完成后处理 |
| steer | 新消息作为追加上下文注入，引导 Agent 调整方向 |

## 后台会话

`/background` 命令在独立会话中运行任务：

- 隔离会话，不影响主聊天
- 继承当前配置
- 完成后结果投递回原聊天

## 工具进度通知

```yaml
display:
  tool_progress: all  # off | new | all | verbose
```

| 值 | 说明 |
|---|---|
| off | 不显示工具进度 |
| new | 仅显示新出现的工具 |
| all | 显示所有工具进度（默认） |
| verbose | 详细输出，包含参数与返回值 |
