## 系统要求

| 平台 | 最低版本 | 说明 |
|------|----------|------|
| macOS | 12+ | 原生支持，沙箱用 Seatbelt |
| Linux | Ubuntu 20.04+ / glibc 2.28+ | 沙箱用 bwrap + seccomp |
| Windows | 通过 WSL2 | 原生 Windows 支持为实验性 |
| 内存 | 4 GB+ | 最低要求 |
| Rust | 1.85++ | 仅从源码构建时需要 |

> **注意**：Windows 原生支持为实验性特性，推荐使用 WSL2 获得完整沙箱功能。

## 安装方式

### 方式一：一键脚本安装（推荐）

**macOS / Linux**：

```bash
curl -fsSL https://chatgpt.com/codex/install.sh | sh
```

**Windows (PowerShell)**：

```powershell
irm https://chatgpt.com/codex/install.ps1 | iex
```

脚本会自动检测平台、下载对应二进制、安装到系统路径。

### 方式二：npm 安装

```bash
npm install -g @openai/codex
```

需要 Node.js 18+。适合已有 Node.js 环境的开发者。

### 方式三：Homebrew 安装（macOS）

```bash
brew install --cask codex
```

适合 macOS 用户，通过 Homebrew 管理更新。

### 方式四：GitHub Releases 手动下载

从 [GitHub Releases](https://github.com/openai/codex/releases) 下载对应平台的二进制文件：

| 平台 | 文件名模式 |
|------|-----------|
| macOS (Intel) | `codex-x86_64-apple-darwin` |
| macOS (ARM) | `codex-aarch64-apple-darwin` |
| Linux (x64) | `codex-x86_64-unknown-linux-musl` |
| Linux (ARM64) | `codex-aarch64-unknown-linux-musl` |

下载后赋予执行权限并放入 PATH：

```bash
chmod +x codex-*
sudo mv codex-* /usr/local/bin/codex
```

### 方式五：从源码构建

需要 Rust 工具链（1.85+）：

```bash
git clone https://github.com/openai/codex.git
cd codex
cargo build --release
# 二进制位于 target/release/codex
```

## 认证配置

Codex CLI 支持两种认证方式：

### 方式一：ChatGPT 登录（推荐）

```bash
codex
# 首次运行会提示登录 ChatGPT 账户
```

通过浏览器 OAuth 流程登录，无需管理 API Key。适合 ChatGPT Plus / Pro / Team 订阅用户。

### 方式二：API Key

```bash
export OPENAI_API_KEY="sk-..."
```

或写入配置文件 `~/.codex/config.toml`：

```toml
model_provider = "openai"
```

API Key 从环境变量 `OPENAI_API_KEY` 读取。

> **选择建议**：有 ChatGPT 订阅用方式一；用 API 计费用方式二。

## CODEX_HOME 环境变量

Codex CLI 的所有配置和数据默认存储在 `~/.codex/` 目录。可通过 `CODEX_HOME` 环境变量自定义：

```bash
export CODEX_HOME="/path/to/custom/codex/home"
```

`CODEX_HOME` 目录结构：

```
~/.codex/
├── config.toml              # 用户级配置
├── AGENTS.md                # 全局 Agent 指令
├── auth.json                # 认证信息
├── sessions/                # 会话历史
├── logs/                    # 日志
└── <profile>.config.toml    # Profile 配置文件
```

## 升级

| 安装方式 | 升级命令 |
|----------|----------|
| 一键脚本 | 重新运行安装脚本 |
| npm | `npm update -g @openai/codex` |
| Homebrew | `brew upgrade --cask codex` |
| GitHub Releases | 下载新版本替换 |
| 源码构建 | `git pull && cargo build --release` |

## 卸载

```bash
# npm
npm uninstall -g @openai/codex

# Homebrew
brew uninstall --cask codex

# 手动
rm /usr/local/bin/codex
rm -rf ~/.codex/
```

## 常见问题

### Q: Windows 原生支持怎么样？

实验性。推荐使用 WSL2 获得完整沙箱功能。原生 Windows 沙箱功能有限。

### Q: 需要 ChatGPT 订阅吗？

不一定。ChatGPT 登录需要订阅（Plus/Pro/Team），API Key 方式按使用量计费，不需要订阅。

### Q: 离线能用吗？

可以配置本地模型（如 Ollama）实现离线使用。需要在 `config.toml` 中配置 `model_provider` 指向本地 provider。
