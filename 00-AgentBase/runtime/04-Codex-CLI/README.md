## 与其他运行时的对比

| 方面 | Codex CLI | Hermes Agent | Claude Code |
|------|-----------|--------------|-------------|
| 开发者 | OpenAI | Nous Research | Anthropic |
| 语言 | Rust | Python + TypeScript | TypeScript |
| 沙箱 | OS 原生（Seatbelt/bwrap） | 容器（Docker） | OS 原生 + 容器 |
| 配置 | TOML | YAML + env | JSON |
| 多平台 | 终端 only | 终端 + 20+ 消息平台 | 终端 + IDE + Web + 桌面 |
| 定位 | 轻量安全本地编码 | 自我进化多平台 Agent | 全表面 Agentic 编码 |

详细架构对比见 [07-架构设计](./07-架构设计.md)。
